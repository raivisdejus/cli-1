# Usage
<!-- usage -->
```sh-session
$ npm install -g scandipwa-cli
$ pwa COMMAND
running command...
$ pwa (-v|--version|version)
scandipwa-cli/0.0.1 linux-x64 node-v8.10.0
$ pwa --help [COMMAND]
USAGE
  $ pwa COMMAND
...
```
